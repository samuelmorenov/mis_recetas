# Mis Recetas

Proyecto personal para guardar mis recetas favoritas.

## ¿Qué es?

Una aplicación web simple para gestionar y consultar recetas. Solo eso. No busco que el código sea bonito, reutilizable o siga ningún patrón especial. Solo funciona y listo.

## Características

- Lista de recetas en la página principal
- Página de detalle para cada receta con ingredientes y pasos de preparación
- Sin dependencias externas, sin servidor necesario
- Funciona directamente abriendo el HTML en el navegador

## Estructura

```
/home/              - Página principal con la lista de recetas
/detalle_receta/    - Página de detalle de cada receta
/recetas/           - Archivos de datos de cada receta
/recetas/imagenes/  - Imágenes de las recetas
```
